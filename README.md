# FBD_population-dynamics

capture_histories.R = query and generate capture histories for state-space model, Fig. 1 creation
state-space_survey-area_seasonal.R = seasonal capture history
state-space_survey-area_annual.R = annual capture history, supplementary
model_run.R = final data formatting & model run, called by state-space_survey-area_seasonal.R and state-space_survey-area_annual.R 
phi_model.R = survival model, called by model_run.R
